# Password Strength Analyzer with Custom Wordlist Generator

## Objective
This project analyzes password strength and generates custom wordlists using user information.

## Features
- Password Strength Checker
- Custom Wordlist Generator
- Leetspeak Conversion
- TXT File Export
- Menu Driven CLI Tool

## Technologies Used
- Python
- zxcvbn
- nltk

## How To Run

Install dependencies:

pip install -r requirements.txt

Run the program:

python main.py

## Output
Generated wordlists are saved in:
wordlist.txt
